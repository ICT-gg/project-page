Frequently asked questions

What's the goal about this experiment?

We wanted to try if we could get a Discord bot (with zero functionality) verified through the Discord developer program. This requires a minimum of 75 invitations.

Who can participate in the experiment?

Everyone with a running and active Discord server can participate.

We don't care what your Server is about or what language you speak.

What commands can I execute?

The Bot has no commands and doesn't even read your messages.

It's pure existence is to test, if we can get a bot with no functionality verified.

How to participate in the experiment?

Just click here and select the server the bot should join.

The bot doesn't come with a role. You need to assign it a role after it joined.

What permissions does the bot need?

The bot doesn't need any permissions at all. Except permission to join the server.

But it would be better if you assign it a role.

What can we do?

We cant't control your Server or do similar things. We will also not read any messages. The only things we can do:

- See how many users are on your server.

- Detect the server-owner. (We will use this Feature to contact the owner if needed.)
